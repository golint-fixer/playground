# metagps

This tool outputs the GPS coordinates of provided images.

## Installation

	go get github.com/mewmew/playground/cmd/metagps

## Usage

	metagps PATH...

## Examples

	metagps "2013-02-17 - 09.09.37.jpg"
	// Output:
	// path: 2013-02-17 - 09.09.37.jpg
	// coord: N 13.410758 E 103.85964
