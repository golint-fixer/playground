# flac2wav

The flac2wav tool converts FLAC files to WAV files.

## Installation

	go get github.com/mewmew/playground/cmd/flac2wav

## Usage

	flac2wav [OPTION]... FILE...

Flags:

	-f (default=false)
		Force overwrite.
